📝 React Todo List App

A simple and clean Todo List built with ReactJS using functional components and React Hooks (`useState`).

🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive and user-friendly

🛠️ Built With
- React.js
- JavaScript (ES6)
- CSS
